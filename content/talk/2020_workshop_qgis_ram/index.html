Apresentar os principais conceitos e técnicas de geoprocessamento utilizando o software QGIS 3.12 (ou versões superiores). No curso será oferecido os principais conceitos em cartografia, GPS, sensoriamento remoto, Sistemas de Informação Geográficos (SIGs), funcionamento do software QGIS, criação, edição e geoprocessamento de dados vetoriais e matriciais (raster), análises geoespaciais simples e elaboração de mapas temáticos
